higher.py, higher2.py - Simple python scripts that calculates the highest number from a list.
They demonstrate the use of "for" loop.

FizzBuzz.py - Simple python script that loops through a range of numbers and prints "Fizz" when the number is divisible by 3, "Buzz" when the number is divisible by 5, "FizzBuzz" when the number is divisible by 15, or it prints the number, if the conditions above aren't met. 
It demonstrates the use of "range()" function.
